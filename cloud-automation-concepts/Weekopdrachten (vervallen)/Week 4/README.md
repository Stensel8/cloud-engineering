# Week 4 - File Storage en Backup in AWS

## Onderwerpen

Deze week verkennen we bestandsopslag en backups in AWS. We automatiseren het provisioneren van meerdere stacks via de AWS CLI en maken een Elastic File System (EFS) aan via CloudFormation.

## Leerdoelen

1. De AWS CLI installeren en configureren voor AWS Academy (us-east-1)
2. CloudFormation-stacks deployen via de AWS CLI
3. Netwerkinformatie ophalen via de AWS EC2 CLI en opslaan in S3
4. Een S3-bucket aanmaken en bestanden uploaden via de CLI
5. Een EFS-bestandssysteem aanmaken via CloudFormation
6. EFS Mount Targets deployen en het bestandssysteem mounten op EC2-instances
7. Redeneren over het gebruik van EFS voor logbestanden en backups
