# Week 1 - Hello AWS & CloudFormation Basis

## Onderwerpen

Deze week maken we kennis met AWS en leren we hoe je infrastructuur automatisch kunt deployen met CloudFormation in plaats van de GUI. We richten een VPC in met een publiek subnet en starten een EC2-instance via een CloudFormation-template.

## Leerdoelen

1. De AWS Academy-omgeving instellen en testen
2. Begrijpen hoe een VPC met een publiek subnet en routering werkt
3. Een CloudFormation-template lezen, begrijpen en uitbreiden
4. Een internet gateway, routetabel, beveiligingsgroep en EC2-instance deployen via CloudFormation
5. De koppeling maken tussen de AWS Console en de onderliggende CloudFormation-configuratie
6. Een nginx-webserver deployen op een Amazon Linux 2 AMI via UserData
