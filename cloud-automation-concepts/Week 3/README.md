# Week 3 - Databases en Storage in AWS

## Onderwerpen

Deze week verkennen we databases en opslag in AWS. We koppelen een PostgreSQL RDS-database aan een webapplicatie via CloudFormation en leren hoe je CloudWatch-alarmen en SNS-notificaties instelt.

## Leerdoelen

1. Een CloudWatch-alarm aanmaken dat triggert bij het stoppen van een EC2-instance
2. Een SNS-topic aanmaken en e-mailnotificaties versturen
3. Een RDS PostgreSQL-database deployen via CloudFormation
4. Een webapplicatie koppelen aan een RDS-database
5. CloudFormation-parameters gebruiken voor dynamische configuratie
6. Een MongoDB-instantie opzetten op een EC2-instance (huiswerk)
